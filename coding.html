    Change Log

    Accredited Wildlife Safe Operator.

    • Completed 'Wise Scheme' training - wisescheme.org
    • The UK national training scheme for minimising disturbance to marine wildlife.
    • Enables me to safely operate a boat, drone & myself around wildlife & marine animals such as seabirds, whales, dolphins, seals & sharks.

    Object detection.

    • Implemented accurate 24/7 object detection using open source tools: Frigate & Coral TPU Acelerator on unraid server.
    • Swap lenses on stock CCTV cameras by use of FPV lenses giving a more useful extra wide field of view.
    • Fine-tune the detection system using python code.
    • Detection of 10 video steams @ 10fps each (or 20streams @ 5fps) for video detection with full res 1080p recorded at 15fps.
    • The coral card uses only 2w of power for object detection! This could be made into a wildlife system running on battery power on location.

    Inspire 3 Night Shoot

    • Built easily removable solution to mount & use additional monitor for a clean view of both the video output & the FPV feed.
    • The new inspire3 is a lovely bit of kit but the new monitors are small compared to the previous inspire2 system's 10" monitors. This means you have less screen 'real estate' on the inspire3.
    • DJI don't provide a mounting solution for external screens or transmitters so everything has to be custom.
    • My solution allows you to still pack the controller away in the stock DJI case.

    Docker

    • Docker allows you to run containers at at an low os hardware level.
    • It is more effient way to share resources on a server compared to VMs.
    • For example multiple docker containers can share a graphics card whereas a VM is required to have it passed though to a single VM instance.

    RTK

    • RTK is a more accurate version of GPS down to 1cm - whereas public free gps has 15-5m accuracy.
    • The inspire2 drone was notorius for drifting up and down about 1-2m.
    • Using our RTK basestation coupled with the inspire3 drone we're able to have cm level precision accuracy so no drone drift. We can also record cm accurate waypoint missions to be repeated in the future. Enabling some exciting creative possibilities!
    • A quick scout on ebay let me pick up the RTK unit from an phantom4 RTK user at a bargain price before anyone really knew they were useful on the inspire3.

    Full Frame

    • The new 8k full frame camera on the inspire3 is awesome. A champion in low light etc.
    • However someone clearly didn't do their maths over at DJI as full frame makes the lenses wider. They released a new wide angle lens when what users need is a new telephoto lens.
    • 85mm is needed ASAP & ideally a 120mm lens too.

    ZFS

    • ZFS is data file system that provides self-healing data protection.
    • Pooled storage
    • Copy-on-write
    • Snapshots
    • Data integrity verification and automatic repair
    • RAID-Z
    • Maximum 16 Exabyte file size
    • Maximum 256 Quadrillion Zettabytes storage.

    unRAID Server

    • Researched, Designed & Built a custom Server for data backup.
    • Qnap are market leaders but I wanted to create something with more custom features & upgradability for the same or less cost.
    • Unraid is a Linux OS that enables flexability. At its core is parity backup system. Meaning you can backup the data of 30disks onto 1 drive. You can also run regular RAID & ZFS configurations.
    • Used a budget friendly chinese 8bay NAS case but changed all fans to noctua silent fans.
    • The server uses a low power 35w intel i9 processor with 8cores & 16threads.
    • 128GB RAM to be able to run docker containers & VM's. 10GB ethernet for fast networking. 8TB of SSDs & 48TB of drives with a HBA LSI SAS card that enables the expansion of virtually limitless drives.

    Wifi Access Points

    • Researched & built a custom wifi access point network in antipation of 1000Mb fibre broadband installation in an old stone house with thick walls.
    • Previously used a wireless mesh network but this was limited to 150Mb/s.
    • Used affordable TPLink hardware and hard wired cat6 cable links.
    • Access points share the same IP subnet so users can seamlessly roam the network.
